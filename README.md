# 🌦️ Martech Weather-Based Campaign Selector

### 🎯 Objetivo del Proyecto
Este sistema demuestra una integración de **Martech Engineering** donde se utiliza una fuente de datos externa (API de clima) para automatizar la toma de decisiones de marketing en tiempo real.

### 🛠️ Tecnologías y Conceptos Aplicados
* **Python 3**: Motor de lógica de negocio.
* **OpenWeather API**: Consumo de datos externos vía HTTP.
* **Seguridad (Dotenv)**: Gestión de credenciales mediante variables de entorno y `.gitignore`.
* **Git/GitHub**: Control de versiones profesional.

### 📂 Estructura del Proyecto
* `main.py`: Script principal con la lógica de segmentación.
* `.env`: (Oculto) Almacena la API Key de forma segura.
* `.gitignore`: Evita la fuga de datos sensibles a repositorios públicos.
